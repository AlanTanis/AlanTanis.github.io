新浪财经行情 API

发表于 2019-06-11 分类于 开发者 Valine：

新浪财经行情的相关 API 可以通过浏览器开发者工具来挖掘、提取。整理如下。

沪深股市

HTTP 请求方式：GET

请求参数说明

PRE 可选前缀 说明
sh 上交所股票（基金）前缀
s_sh 上交所股票（基金）前缀，省略五档报价等信息
sz 深交所股票（基金）前缀
s_sz 深交所股票（基金）前缀，省略五档报价等信息

SYMBOL 为股票代码。如：000001，为上证指数。
